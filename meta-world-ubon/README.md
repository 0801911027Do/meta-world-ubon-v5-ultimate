# 🌏 Meta World — อุบลราชธานี
### Senior Project | วิศวกรรมคอมพิวเตอร์และปัญญาประดิษฐ์

---

## 🚀 วิธีรันโปรเจค (ทำครั้งแรกครั้งเดียว)

### ต้องติดตั้งก่อน:
- [Node.js LTS](https://nodejs.org) 
- [Git](https://git-scm.com)

### ขั้นตอน:

**1. เปิด Terminal / Command Prompt ในโฟลเดอร์นี้**

**2. ติดตั้ง dependencies:**
```bash
npm install
```

**3. รันโปรเจค:**
```bash
npm run dev
```

**4. เปิดบราวเซอร์ไปที่:**
```
http://localhost:5173
```

---

## 📦 Deploy ขึ้น Vercel (ฟรี)

```bash
npm install -g vercel
vercel
```
แล้วกด Enter ตามขั้นตอน จะได้ลิงก์ออนไลน์ทันที
